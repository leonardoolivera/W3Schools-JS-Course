Guia de Uso do GitHub com Visual Studio Code

1. Instalar o Git e o VSCode

Baixe e instale o Git aqui.
Baixe e instale o VSCode aqui.

2. Configurar o Git no VSCode

  1. Abra o VSCode.
  2. Abra o terminal integrado (View > Terminal ou Ctrl + `).
  3. Configure seu nome de usuário e e-mail do Git:
    4. git config --global user.name "Seu Nome"
git config --global user.email "seuemail@exemplo.com"

3. Criar uma conta no GitHub

Visite github.com e siga os passos para criar sua conta e verificar seu e-mail.

4. Criar um novo repositório no GitHub

  1. Logue na sua conta do GitHub.
  2. Clique em "New" na seção de repositórios.
  3. Preencha os detalhes do repositório.
  4. Escolha se será público ou privado e marque a opção para criar um README inicial.
  5. Clique em "Create repository".

5. Clonar o repositório no VSCode

  1. Copie o URL do repositório no GitHub.
  2. No VSCode, abra a Command Palette (Ctrl + Shift + P).
  3. Digite "Git: Clone" e cole o URL quando solicitado.
  4. Selecione onde o repositório deve ser clonado no seu computador.

6. Fazer alterações e commitar

  1. Faça alterações nos arquivos no VSCode.
  2. Salve as alterações nos arquivos no VSCode.
  3. Abra a aba de Source Control (ícone de branches no lado esquerdo).
  4. Adicione os arquivos alterados à staging area clicando no sinal de "+".
  5. Digite uma mensagem de commit no campo "Message" e clique no ícone de "check" para commitar.

7. Enviar as alterações para o GitHub

Clique no ícone de "synchronize changes" (canto inferior esquerdo) para fazer push das alterações para o GitHub.

Aceite tudo e coloque o código ou entre com sua conta github do jeito que achar melhor

8. Gerenciar branches e Pull Requests

  1. Para criar um novo branch, abra a Command Palette e digite "Git: Create Branch".
  2. Para alternar entre branches, clique no nome do branch na parte inferior e selecione o branch desejado.
  3. Para abrir um Pull Request, vá para o GitHub, acesse o repositório, clique em "Pull requests" > "New pull request", escolha o branch e siga os passos para criar o PR.

9. Mesclar um Pull Request no GitHub

No GitHub, vá para a página do pull request, clique em "Merge pull request" e confirme.
